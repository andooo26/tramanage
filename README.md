# tramanage
- Webアプリから音楽ファイルの管理ができる

## 機能

### ディレクトリビューワ (DirectoryView)
- 選択したディレクトリをツリー階層で確認
- 特定の拡張子以外のファイルを一覧表示
- 隠しファイルの一括削除

### 差分を比較 (DiffView)
- 2つのディレクトリを選択し、その差分を表示
- JPG, PNGは検索対象でない
- 差分があればコピーを行う事も可能

### 音源を変換 (AudioConv)
- 指定したディレクトリ内のmp3以外の音源をmp3へ変換する機能
- 変換したファイルは/convertedへ格納される
- 元ファイルは削除されない
